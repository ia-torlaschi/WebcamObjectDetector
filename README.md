Webcam Object Detector
Proyecto para detectar objetos cotidianos en tiempo real usando la webcam, YOLOv8 y Python. Muestra los nombres de los objetos como subtítulos.

Setup
Clona el repositorio.
Crea un entorno virtual: python -m venv venv
Activa el entorno: .\venv\Scripts\Activate.ps1 (en PowerShell)
Instala las dependencias: pip install -r requirements.txt
Ejecuciónbash
python main.py
